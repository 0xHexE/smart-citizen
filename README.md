# Prerequisite

[![Greenkeeper badge](https://badges.greenkeeper.io/httpsOmkar/smart-citizen.svg)](https://greenkeeper.io/)

* node-js
* angular-cli

# Installation
  * Install nodejs
  * `npm i -g @angular/cli` or `yarn add global @angular/cli`
  * cd to folder
  * `npm i` or `yarn`
  * `ng serve`

You need to init you own envornment.ts file
